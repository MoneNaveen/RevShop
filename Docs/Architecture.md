___
### 🔄 DATA FLOW
```text
User Input
   ↓
Console Menus (RevShopApplication)
   ↓
Service Layer (Business Rules & Validation)
   ↓
DAO Layer (JDBC)
   ↓
MySQL Database
   ↑
Formatted Response back to User
```
___
### 🏗️ DEPENDENCY FLOW (Strict)
```text
RevShopApplication
        ↓
     Service Layer
        ↓
       DAO Layer
        ↓
 DBConnectionUtil
        ↓
      MySQL
```
